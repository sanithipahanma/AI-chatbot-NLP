# 🤖 AI Chatbot for Retail using NLP

This project is an AI-based chatbot developed using Python and basic Natural Language Processing (NLP) techniques. It processes user input, cleans text data, and generates responses based on predefined patterns.

## 🚀 Features
	•	Text preprocessing (cleaning & normalization)
	•	Stopword removal
	•	Tokenization
	•	Lemmatization
	•	Keyword-based response system
	•	Data storage for future improvements (Google Drive / Excel)

## 🛠️ Technologies Used
	•	Python
	•	NLTK (Natural Language Toolkit)
	•	Google Colab
	•	Regular Expressions (re)
	•	TextBlob

## 🧠 How It Works
	1.	User input is cleaned (removing symbols and extra spaces)
	2.	Stopwords are removed
	3.	Text is tokenized into words
	4.	Lemmatization is applied to normalize words
	5.	The chatbot matches keywords and returns relevant responses
